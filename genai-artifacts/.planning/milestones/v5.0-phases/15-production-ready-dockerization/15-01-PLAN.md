<task type="auto">
  <name>Task 4: Create multi-stage Dockerfile for Backend</name>
  <files>backend/Dockerfile</files>
  <action>
    Create a multi-stage `Dockerfile` in the `backend/` directory:
    
    Stage 1: Build
    - Use `node:20-alpine`.
    - Install build dependencies (openssl, libc6-compat).
    - Copy package files and install all dependencies (`npm ci`).
    - Copy `prisma/schema.prisma` and run `npx prisma generate`.
    - Copy source code and run `npm run build`.
    - Run `npm prune --production` to remove dev dependencies.
    
    Stage 2: Runtime
    - Use `node:20-alpine`.
    - Install `python3` and `py3-pip`.
    - Install `casparser` via pip (`pip install --no-cache-dir casparser --break-system-packages`).
    - Set environment variables: `NODE_ENV=production`, `PYTHON_PATH=python3`.
    - Copy production `node_modules` (including generated Prisma client), `dist`, `prisma/schema.prisma`, `scripts/`, and `package.json` from Stage 1.
    - Create `uploads` directory and ensure it's owned by the `node` user.
    - Switch to non-root user `node`.
    - EXPOSE 3001.
    - CMD ["node", "dist/index.js"].
  </action>
  <verify>
    docker build -t portfolio-tracker-backend ./backend
  </verify>
  <done>Backend Dockerfile created and image builds successfully.</done>
</task>
