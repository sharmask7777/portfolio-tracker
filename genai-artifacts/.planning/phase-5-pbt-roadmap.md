This roadmap outlines the steps to introduce institutional-grade mathematical verification using Property-Based Testing (PBT) and synthetic data generation.

2.  **Task 5.2: Property-Based Testing (PBT) Foundation** (`.planning/phase-5-plans/5-2-PLAN.md`)
    *   Install `fast-check` and create data arbitraries.
3.  **Task 5.3: PBT for Core Math & Taxation** (`.planning/phase-5-plans/5-3-PLAN.md`)
    *   Verify XIRR stability and exact tax loss set-off invariants.
4.  **Task 5.4: PBT for Analytics & Aggregation** (`.planning/phase-5-plans/5-4-PLAN.md`)
    *   Verify Family aggregation math and X-Ray weighting percentages.
