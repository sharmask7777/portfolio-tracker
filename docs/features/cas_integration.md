# CAS Integration

## Overview
The application supports seamless importing of Consolidated Account Statements (CAS) generated by RTAs such as CAMS and Karvy. This is a core feature that enables automated tracking of mutual fund transactions.

## casparser Integration
We utilize a Python script based on `casparser` to extract structured data from password-protected PDF CAS files.
- The user uploads a CAS PDF and provides the password.
- The background worker executes the Python `casparser` utility to parse the file into JSON format.
- The JSON output contains folios, schemes, and individual transactions which are then ingested into our database for further processing and analytics.
