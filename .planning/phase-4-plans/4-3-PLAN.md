# Phase 4, Task 3: Portfolio Health Checks & Goal Tracking

## Objective
Add "Smart Insights" to the dashboard to help users maintain a healthy portfolio and stay on track for financial goals.

## Given
*   Detailed portfolio structure (Sector, Cap, Overlap).
*   User investment data.

## When
1.  Implement a "Health Check" engine:
    *   Detect high concentration risk (e.g., >30% in one sector).
    *   Detect high stock overlap risk.
    *   Detect "Asset Drift" (deviation from a target 60/40 or custom allocation).
2.  Implement a "Goal Tracker":
    *   Allow users to define goals (e.g., "Retirement", "House Fund") with a target amount and date.
    *   Link specific portfolios or folios to these goals.
    *   Visualize "Progress to Goal" % and "Estimated Shortfall".
3.  Create an "Insights" sidebar on the dashboard to display these alerts.

## Then
*   The system moves from passive tracking to proactive financial advisory.
