# Phase 2, Task 4: Advanced Visualizations (Frontend)

## Objective
Create the UI for Portfolio X-Ray and Stock Intersection.

## Given
*   Frontend (React/TypeScript) with Recharts.
*   Backend `GET /api/portfolio/xray` and overlap data.

## When
1.  Create a new `XRay` tab/view in the frontend.
2.  Implement a **Sector Treemap** or **Sunburst Chart** (using Recharts or a specialized library) for sector exposure.
3.  Implement a **Market Cap Distribution** bar chart.
4.  Build a **Stock Intersection Table** showing top 10 individual stock exposures and which funds contribute to them.
5.  Add a "Fund Overlap" matrix to compare any two mutual funds for similarity.

## Then
*   The user should have a clear, visual understanding of their portfolio's deep structure, beyond just "which fund is up or down."
