# Phase 3, Task 4: Tax Dashboard & ITR Reporting

## Objective
Visualize tax metrics and generate exportable reports.

## Given
*   Realized and Unrealized gain data from previous tasks.

## When
1.  Create a "Tax" tab in the frontend.
2.  Build a **LTCG Exemption Tracker**: A progress bar showing how much of the ₹1.25L limit is used.
3.  Implement a **Realized Gains Summary** table (grouped by asset and tax type).
4.  Develop an "Export Capital Gains Report" (CSV/Excel) in a format similar to MProfit or CAMS, suitable for ITR filing.
5.  Add visual "Tax Tags" (ST/LT) to the main holdings table.

## Then
*   The user has a single place to manage their tax liabilities and prepare for tax season.
