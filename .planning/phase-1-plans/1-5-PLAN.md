# Phase 1, Task 5: Initial Visual Dashboard (MVP)

## Objective
Build the first version of the dashboard to display the parsed and analyzed portfolio.

## Given
*   Frontend (React/TypeScript).
*   Backend APIs for portfolio data and performance metrics.

## When
1.  Design a clean, data-dense layout using Vanilla CSS.
2.  Implement a "Summary" card showing: Total Net Worth, Total Gain/Loss, Overall XIRR.
3.  Build a "Holdings Table" listing all schemes with current value, units, NAV, and returns.
4.  Integrate a charting library (Recharts) to show Portfolio Value vs. Invested Capital over time.
5.  Implement a "Transaction History" view with filtering by asset and date.
6.  Implement basic Dark/Light mode toggle.

## Then
*   The user should be able to see a visual summary of their portfolio immediately after a successful import.
*   The UI should be responsive and follow the established "data-dense SaaS" aesthetic.
