  <name>Task 1: Implement HistoryService.getPortfolioStats</name>
  <files>backend/src/services/history.service.ts</files>
  <action>
    Extend the `HistoryService` class with a static method `getPortfolioStats(portfolioIds: string[])`.
    - It should query the `PortfolioHistory` table for all records matching the provided `portfolioIds`.
    - Aggregate values by date (if multiple IDs are provided, e.g., for consolidated view).
    - Identify "All-Time High Corpus": The maximum `value` and its `date`.
    - Identify "Maximum Invested": The maximum `investedAmount` and its `date`.
