# Context: Project Skeleton & Environment Setup

## Task Overview
Initialize the monorepo structure for the Portfolio Tracker project, including a React frontend, Node.js backend, and Dockerized infrastructure.

## Requirements
- Git initialization.
- Backend: Node.js, Express, TypeScript, basic middleware.
- Frontend: React (Vite), TypeScript.
- Tooling: ESLint, Prettier.
- Infrastructure: Docker Compose with PostgreSQL and Redis.
- Validation: Health-check endpoints, linting/formatting checks.

## Tech Stack
- Frontend: React, TypeScript, Vite.
- Backend: Node.js, Express, TypeScript.
- Database: PostgreSQL.
- Cache: Redis.
- Orchestration: Docker Compose.
